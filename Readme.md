# Пет проект "Task Manager" для развития навыков

## На данный момент решения состоит из следующих модулей:

	1. TM.Domain - содержит предметные модели и расширения для их валидации.
  
	2. TM.Persistence - содержит контекст для работы с БД
  
	3. TM.Persistence.Migrator.CLI - позволяет произвести инициализацию и миграцию БД 
  	под несколько разных провайдеров.
    
	4. TM.Application - содержит логику (сервисы, репозитории, петтерны) для работы приложения.
  
	5. TM.WebServer - сервер(служба) со сваггером.
  
	6. TM.Application.Tests - Unit тесты с моками.
